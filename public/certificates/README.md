# Certificates Folder

Yahan aap apne certificate images add kar sakte hain.

## Instructions:

1. Apne certificate images ko is folder mein copy karein
2. `Certificate.tsx` file mein `certificateImages` array mein entry add karein:

```typescript
{
  id: 'cert-1',
  title: 'Certificate Name',
  image: '/certificates/your-certificate-image.jpg',
  issuer: 'Issuing Organization',
  date: '2024'
}
```

## Recommended Image Format:
- Format: JPG or PNG
- Size: 800x600px ya similar aspect ratio
- File size: 500KB se kam (optimize karein)

## Example:
- `/certificates/tops-technologies-cert.jpg`
- `/certificates/java-certification.png`
- `/certificates/react-certificate.jpg`
